# Gather Paginated List of Equity Orders

Grab a list of orders.

**URL** : `/orders/`

**Method** : `GET`

**Auth required** : YES

**Permissions required** : None

**Query parameters**

    - `updated_at[gte]` - greater than or equal to a date; timestamp or ISO 8601
    - `updated_at[lte]` - less than or equal to a date; timestamp or ISO 8601
    - `instrument` - equity instrument URL
    - `cursor` - pagination placeholder

**Request samples**

```
curl -v https://api.robinhood.com/orders/ \
   -H "Accept: application/json" \
   -H "Authorization: Token a9a7007f890c790a30a0e0f0a7a07a0242354114"
```
